Lumis Experience Platform

UX/UI Front End

LumisXP, a evolution of Lumis Portal, is a Java based Customer Experience Platform, with DAM, CMS, access control and CX modulues, where you can build sites, intranets and systems.

In respect to the Lumis Experience Platform, I've been responsible for all decisons, UX and product-wise, together with the development team. I've also consistently done front end development for the project, mainly in the newer interfaces which are using Vue.js and a SPA architecture.
